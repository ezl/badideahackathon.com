The Best Way To Send Me Your Ideas

Format

When you send me an email, a computer program will automatically read and process the email. In order to make sure it's understood and processed correctly, you should follow the following format:

  1. Tagline

    The first line of the email should be a short description of your idea.

    Think of this as you would the title of a blog post. It should be short, specific, and descriptive.

    It might be what you would tell someone if you only had 15 seconds to convey an idea to him/her or what you would type in a tweet if only given 140 characters to explain yourself.

  2. A blank line

    After the tagline, include a blank line to separate from the rest of the email. Everything before the FIRST blank line will be treated as the "tag line". Everything AFTER will be treated as a detailed description.

    The 2nd, 3rd, and subsequent blank lines are just considered part of the detailed description.

  3. Detailed description (optional)

    This is optional. If you want to expand on the tag line, write down more information about the idea, or explain it in detail -- this is the place to do it.

    If the tag line is like the title of a blog post, this is like the main body of the blog post. Be as descriptive as you like.

    Additionally, for nerds: The parser reads this as Markdown, so feel free to add formatting via Markdown.

Here's a sample email:

A site for people to save and share ideas <-- tagline
                                          <-- blank line
It would be really great if there was a   <-- detailed description
site that helped me get in the habit of coming up with new ideas. Every day, the site should send me an email that says something like, "Hi Eric! What's your idea today?".
                                          <-- subsequent blank lines are ok
Then the site automatically processes and categorizes the idea and creates a log of all my ideas.

I can update, edit, or improve on the ideas as they mature. Other people can also comment or add feedback to my ideas. The site should automatically also match different ideas up that are similar, then notify me when other similar ideas are created or start getting new activity.
